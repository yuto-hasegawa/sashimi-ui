# Troubleshooting

## My styles aren’t showing up

Please check the following:

- You need to load both the **bundled CSS** and a **theme CSS** file.
- If you are using the **prefixed** style sheet, make sure every class name starts with `sui-`.

For detailed usage, see the <a class="link" href={`${import.meta.env.BASE_URL}getting-started`}>Getting Started</a> page.

## Icon color doesn’t change

sashimi UI doesn’t support changing icon colors.
Color appearance in this documentation examples relies on the SVG’s own styling capabilities.
For details, see the <a class="link" href={`${import.meta.env.BASE_URL}components/icon`}><code>.icon</code></a> page.
