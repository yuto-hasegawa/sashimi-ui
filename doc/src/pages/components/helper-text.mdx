# Helper Text

`.helper-text` makes elements **look like** a form helper text.

## Applicable Elements

- `p`
- any other text elements

## Usage

<SamplePane>
<fieldset class="fieldset">
<input type="datetime-local" class="input" />
<p class="helper-text">This is a helper text.</p>
</fieldset>
</SamplePane>

```html
<fieldset class="fieldset">
  <input type="datetime-local" class="input" />
  <p class="helper-text">This is a helper text.</p>
</fieldset>
```

## States

<SamplePane>
<fieldset class="fieldset">
<input type="datetime-local" class="input error" />
<p class="helper-text error">Date is required.</p>
</fieldset>
</SamplePane>

```html
<fieldset class="fieldset">
  <input type="datetime-local" class="input error" />
  <p class="helper-text error">Date is required.</p>
</fieldset>
```

## React component

React component `<HelperText/>` is available.  
It shows an error message when `error` is specified.

It inherits the error property from a <a href={`${import.meta.env.BASE_URL}components/fieldset/`}>`Fieldset`</a> component and automatically applies the error style.
