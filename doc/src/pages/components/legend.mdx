# Legend

`.legend` is for html legend element.

## Applicable Elements

- `legend`

## Usage

<SamplePane>
<fieldset class="fieldset">
  <legend class="legend">Name</legend>
  <input class="input" placeholder="Yuto Hasegawa" />
</fieldset>
</SamplePane>

```html
<fieldset class="fieldset">
  <legend class="legend">Name</legend>
  <input class="input" placeholder="Yuto Hasegawa" />
</fieldset>
```

## States

<SamplePane>
<fieldset class="fieldset">
  <legend class="legend error">Name</legend>
  <input class="input error" value="123456" />
</fieldset>
</SamplePane>

```html
<fieldset class="fieldset">
  <legend class="legend error">Name</legend>

</fieldset>
```

## React component

React component `Legend` is available.  
It inherits the error property from a [`Fieldset`](/components/fieldset) component and automatically applies the error style.
